# JPYC Contract Interaction Sample

このプロジェクトはSepoliaテストネット上のJPYCコントラクトと対話するためのサンプル実装です。

## セットアップ

1. **リポジトリのクローン**
   ```bash
   git clone <このリポジトリのURL>
   cd jpyc
   ```

2. **依存関係のインストール**
   ```bash
   pnpm install
   ```

3. **環境変数の設定**
   ```bash
   cp .env.example .env
   ```

## 使用方法

`src/index.ts`の実行：

```bash
pnpm start
```
